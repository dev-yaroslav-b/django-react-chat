# Chat room created with Django and React
Simple chat using:
1. Django + Django Channels
2. React 
3. PostgreSQL

##Installation
Clone project:
```
git clone https://github.com/dev-yaroslav-b/django-react-chat.git
```
To run backend:
change directory
```
cd backend/
```
and run
```
docker-compose up
```
To run frontend:
change directory
```
cd frontend/
```
and run
```
npm i
npm start
```
